Korean-Romanizer-KoreanName

Specialised for Korean names, providing a variety of name combinations.

About

This library focuses on Korean personal names and generates a variety of romanized combinations for them.

Features

Accepts Korean names (in Hangul) and produces romanized formats.

Supports multiple combination styles (e.g., surname-first, given-name separate, various spacing).

Useful for passport applications, resume name formatting, international systems, etc.

Getting Started

Clone the repository

git clone https://github.com/caiseye/Korean-Romanizer-KoreanName.git


Add it to your project dependencies (as appropriate for your build tool).

Use the API (example):

// Example usage in Java
Romanizer romanizer = new KoreanNameRomanizer();
String result = romanizer.romanize("박지성");
System.out.println(result);  // e.g., “Park Jisung”, “Bak Ji-seong”, etc.


Configure style options, if available (surname style, spacing, hyphenation, etc).

한국어
Korean-Romanizer-KoreanName

한국인의 이름에 특화된, 다양한 이름 조합(로마자 표기)을 제공하는 라이브러리입니다.

개요

한글로 된 한국인의 이름을 입력하면, 여러 가지 로마자 표기 조합을 생성해주는 라이브러리입니다.

주요 기능

한국어 이름(한글) 입력 → 로마자 표기 출력

다양한 조합 스타일 지원 (예: 성 먼저, 이름 따로, 띄어쓰기/하이픈 선택 등)

여권 신청서, 이력서 이름 표기, 해외 시스템 연동 등에 활용 가능

사용 시작

저장소 복제

git clone https://github.com/caiseye/Korean-Romanizer-KoreanName.git


프로젝트에 의존성으로 추가 (빌드 툴에 따라 설정)

API 사용 예시:

// Java 예시
Romanizer romanizer = new KoreanNameRomanizer();
String result = romanizer.romanize("박지성");
System.out.println(result);  // 예: “Park Jisung”, “Bak Ji-seong” 등


스타일 옵션 설정 가능 (성씨 표기 방식, 띄어쓰기/하이픈 여부 등)
